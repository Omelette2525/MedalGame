遊び方
左クリックでメダルを消費します。
長押しすると連続で消費できます。
赤いボックスにメダルが触れるとスロットが始まります。
ボールを落とすとステーションチャレンジ(SC)に発展します。

SCの説明
ボールがクルーンの穴に入った時点でのポケットが配当となります。
配当はジャックポットチャレンジ(JPC)とメダルの2種類が存在します。
JPCポケットの数に応じてLevelが決まります。レベルが高いほどJPCが有利になります。

BrightJPCの説明
一球勝負の抽選です。JPと書かれているポケットに入ればジャックポット獲得です。
Levelが上がるとポケットの内容が豪華になっていきます。

ShadowJPCの説明
outポケットに入るまで続く抽選です。JPと書かれているポケットに3回以上入るとボーナスを獲得できます。
Levelが上がるとoutに入ってもセーフになるターン数が増えます。
